# Bars AI — Платформа обучения

Геймифицированная AI-платформа для интерактивного обучения с менторством, спринтами, coding-упражнениями и системой достижений.

**Production:** https://app-production-ebd5.up.railway.app
**Admin:** https://admin-production-1563.up.railway.app
**GitHub:** https://github.com/AzamatRaimbekov/bars-ai
