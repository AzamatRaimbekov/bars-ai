# PathMind V2 — Design Spec

**Дата:** 2026-04-02
**Проект:** PathMind — AI-powered gamified learning platform
**Цель:** Добавить 20 новых фич для создания продукта уровня Duolingo + Coursera с AI-персонализацией

---

## 1. Целевая аудитория

Смешанная: студенты, сотрудники на корпоративном обучении, самоучки, меняющие профессию. Платформа должна быть одинаково привлекательна для всех.

## 2. Вдохновение

- **Duolingo:** геймификация, лиги, стрики, соревновательность
- **Coursera/Stepik:** структурированные курсы, сертификаты, аналитика прогресса

## 3. Принципы дизайна

- Визуальный wow-эффект: анимации, микроинтеракции, полированный UI
- Функциональная уникальность: AI-возможности, которых нет у конкурентов
- Эмоциональная привязка: ощущение прогресса, желание возвращаться каждый день

---

## 4. Бэкенд и архитектура данных

### Стек

- **Backend:** FastAPI (Python)
- **База данных:** PostgreSQL
- **Кеш/Лидерборды/Сессии:** Redis
- **API-стиль:** REST + WebSocket (PvP-дуэли, real-time лидерборд)
- **Аутентификация:** JWT + refresh tokens, OAuth (Google/GitHub)

### Выбор FastAPI обоснован:

- Простая интеграция с Claude API через Anthropic Python SDK
- Быстрое прототипирование — Pydantic-модели, автогенерация документации
- Async из коробки — подходит для AI-запросов и WebSocket
- На текущем этапе Go — преждевременная оптимизация

### Основные сущности БД

| Сущность | Описание |
|----------|----------|
| `User` | Профиль, настройки, OAuth |
| `Progress` | Пройденные уроки, ноды, XP-история |
| `Achievement` | Бейджи, стрики |
| `LeaderboardEntry` | Еженедельные очки по лигам |
| `DailyQuest` | 3 ежедневных задания, генерируются AI |
| `WeeklyChallenge` | Еженедельное задание |
| `Certificate` | Выданные сертификаты с ID верификации |
| `Friendship` | Связи между пользователями |
| `CodeReview` | Запросы на AI-ревью и результаты |
| `SpacedRepetitionCard` | Карточки с интервалами повторения (SM-2) |
| `Notification` | In-app уведомления |
| `Gem` | Транзакции внутренней валюты |
| `ActivityFeed` | Лента достижений |

## 5. Лиги и лидерборды

### Система лиг (5 уровней)

Бронза → Серебро → Золото → Платина → Бриллиант

- Каждая лига = 30 участников
- Еженедельный цикл: понедельник → воскресенье
- Топ-10 повышаются, нижние 5 понижаются
- Очки лиги = XP, заработанный за текущую неделю

### Лидерборд

- Аватар, имя, XP за неделю, позиция, тренд (вверх/вниз)
- Анимация при смене позиции
- Уведомления: "Ты поднялся на 3-е место!" / "Осторожно, зона вылета!"
- Доступен в Sidebar или на Dashboard

### Награды за лиги

- Повышение лиги = уникальный бейдж + бонус XP
- Топ-1 за неделю = специальная анимация + эксклюзивный бейдж
- Бриллиантовая лига = особый фрейм аватара

### Анти-читинг

- XP только за реальные действия (уроки, квизы, симулятор)
- Rate limiting на бэкенде
- Подозрительные паттерны → ручная проверка

## 6. Ежедневные квесты и стриковые бонусы

### Ежедневные квесты (3 в день)

Генерируются AI в 00:00 под уровень пользователя:

| Сложность | XP | Пример |
|-----------|----|--------|
| Лёгкий | 25 | "Пройди 1 флеш-карточку" |
| Средний | 50 | "Ответь правильно на 5 вопросов квиза" |
| Сложный | 100 | "Пройди симуляцию собеседования" |

- Прогресс-бар: 0/3 → 3/3 с анимацией
- Бонус за все 3: +50 XP + значок "Полный комплект"

### Еженедельный челлендж

- Одно задание на неделю (например, "Заверши 3 ноды на роадмапе")
- Награда: 500 XP + редкий бейдж

### Стриковые бонусы

| Стрик | Множитель XP | Бонус |
|-------|-------------|-------|
| 1-6 дней | x1.0 | — |
| 7 дней | x1.2 | Бейдж "Неделя огня" |
| 14 дней | x1.3 | Бейдж "Две недели" |
| 30 дней | x1.5 | Бейдж "Месяц силы" + фрейм аватара |
| 60 дней | x1.7 | Бейдж "Несокрушимый" |
| 100 дней | x2.0 | Бейдж "Легенда" + уникальный титул |

### Streak Freeze

- 1 бесплатная заморозка в неделю
- Дополнительные — за gems (20 gems)

## 7. AI-адаптивность и AI-ревью кода

### Адаптивная сложность

- После каждого квиза AI оценивает процент правильных ответов
- < 60% → понижение сложности, предложение повторить
- 60-85% → оптимальная зона, продолжаем
- \> 85% → повышение сложности, пропуск базовых тем
- `difficulty_score` хранится per пользователь per направление
- AI-ментор адаптирует объяснения под уровень

### AI-генерация персональных уроков

- Пользователь застрял → AI генерирует дополнительный мини-урок
- Кнопка "Объясни по-другому" — AI перефразирует материал
- Генерация задач под слабые места

### AI-ревью кода (Frontend)

- Страница `/code-review` с редактором и подсветкой синтаксиса
- Пользователь вставляет код → AI анализирует:
  - Корректность
  - Чистота кода и best practices
  - Производительность
  - Безопасность
- Оценка A-F, построчные комментарии, рекомендации
- История ревью — отслеживание прогресса качества кода

### AI Daily Tip (улучшение)

- Совет основан на аналитике: что учил вчера, где ошибался

## 8. Spaced Repetition и мини-игры

### Spaced Repetition (алгоритм SM-2)

- Каждый пройденный урок создаёт карточки для повторения
- Интервалы: 1д → 3д → 7д → 14д → 30д → 90д
- При ошибке — интервал сбрасывается
- Страница `/review`: "Сегодня на повторение: 12 карточек"
- Уведомление на Dashboard
- XP за повторение: 10 XP за карточку

### Мини-игры

**1. Typing Race**
- Печатай код на скорость (для Frontend)
- Отрывки реального кода (HTML, CSS, JS)
- Таймер + точность = очки
- Лидерборд по скорости печати

**2. Code Puzzles**
- Перетаскивай строки кода в правильном порядке
- 3 уровня: 5, 10, 15 строк
- Таймер — быстрее = больше XP

**3. Quiz Battle (PvP)**
- Матч 1v1 в реальном времени (WebSocket)
- 10 вопросов, кто быстрее и точнее
- Matchmaking по лиге и уровню
- Победа = 75 XP, поражение = 25 XP

**4. Term Hunter**
- Найди правильное определение среди 4 вариантов
- 20 вопросов, 10 секунд каждый
- Комбо-множитель за серию правильных

**Доступ:** страница `/games`, быстрый доступ с Dashboard, ежедневные квесты могут включать мини-игры

## 9. Сертификаты и аналитика прогресса

### Сертификаты

- Выдаются при завершении секции (каждые 5 нод) и всего направления
- **Модульный:** "Основы HTML и CSS завершены"
- **Направления:** "Frontend-разработчик"
- Содержимое: имя, направление, дата, уникальный ID
- PDF-экспорт с дизайном PathMind
- Публичная верификация: `/verify/CERT-XXXX`
- Шаринг в LinkedIn / соцсети

### Аналитика (страница `/analytics`)

**Heatmap активности:**
- Календарь за последний год (как GitHub)
- Цвет = интенсивность учёбы
- Hover: "15 марта — 3 урока, 250 XP"

**Графики:**
- XP за неделю / месяц / всё время (line chart)
- Распределение по типам активности (pie chart)
- Прогресс по роадмапу: % завершения (bar chart)

**Слабые зоны:**
- AI анализирует квизы → находит темы с низким %
- Рекомендации: "Повтори: CSS Flexbox, промисы"
- Ссылка на повторение

**Статистика профиля:**
- Всего часов обучения
- Среднее время сессии
- Лучший стрик
- Количество сертификатов
- Позиция в лиге

## 10. Социальные фичи

### Публичный профиль (`/user/username`)

- Аватар + фрейм (зависит от лиги/достижений)
- Титул ("Легенда", "Бриллиантовый мастер")
- Направления и прогресс
- Витрина бейджей (5 выбранных)
- Heatmap активности
- Сертификаты

### Друзья

- Поиск по имени / уникальному коду
- Запрос дружбы → принять / отклонить
- Лента друзей на Dashboard: "Алия завершила модуль CSS"
- Лидерборд среди друзей

### Система уведомлений

- Колокольчик в TopBar
- Типы:
  - "Друг обогнал тебя в лиге!"
  - "Ты в зоне вылета"
  - "Новый ежедневный квест"
  - "12 карточек на повторение"
  - "Получен сертификат!"
- Push-уведомления через Service Worker (опционально)

### Лента достижений (Activity Feed)

- Глобальная лента достижений
- Фильтр: друзья / моя лига / все
- Реакции: "огонь" / "аплодисменты"

## 11. Внутренняя валюта и монетизация

### Gems

**Заработок:**
- Повышение лиги: 50 gems
- Топ-3 за неделю: 30/20/10 gems
- Еженедельный челлендж: 25 gems
- Первый вход за день: 5 gems

**Траты:**
- Streak Freeze (доп.): 20 gems
- Кастомизация аватара: 30-100 gems
- Подсказка в квизе: 10 gems
- Доп. жизни в мини-играх: 15 gems

### Монетизация (на будущее)

**Freemium модель:**
- Бесплатно: 2 направления, 3 квеста/день, базовый AI-ментор
- Premium: все направления, безлимитный AI, расширенная аналитика, без рекламы

Или полностью бесплатно — если проект образовательный/некоммерческий.
