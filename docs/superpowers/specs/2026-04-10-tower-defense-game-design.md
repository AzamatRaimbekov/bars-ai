# Tower Defense — обучающая игра

**Дата:** 2026-04-10
**Статус:** Утверждён

## Обзор

Обучающая Tower Defense игра, встроенная в курсы как step type `"tower-defense"`. Игрок отвечает на вопросы из урока чтобы заработать монеты, тратит их на башни и защищает базу от волн врагов. Дизайн — Friendly Road: скруглённые блоки, эмодзи, яркие цвета на тёмном фоне. Рендер через HTML5 Canvas.

## Игровая механика

### Цикл одной игры

1. **Фаза вопросов (QUESTIONS)** — 3-5 вопросов из steps текущего урока (quiz, true-false, type-answer, fill-blank). Правильный ответ = +30 монет, неправильный = 0. Таймер 15 секунд на вопрос.
2. **Фаза строительства (BUILD)** — игрок видит поле, тратит монеты на башни и апгрейды. Кнопка "Начать волну".
3. **Фаза боя (BATTLE)** — враги идут по дороге, башни стреляют автоматически. Враг дошёл до конца = -1 жизнь.
4. **Повтор** — следующая порция вопросов → строительство → бой.

Количество циклов (волн) = ceil(количество_вопросов / 4).

### Условия

- 3 жизни на всю игру
- Выжил все волны = step пройден (`onAnswer(true)`)
- Потерял все жизни = поражение (`onAnswer(false)`)
- Неиспользованные монеты переносятся между волнами

## Враги

Враги — цветные скруглённые блоки с эмодзи. Визуал: roundRect + эмодзи по центру + полоска HP сверху + тень. При уроне — мигание белым 100ms.

| Враг | Эмодзи | Цвет | HP | Скорость | Размер | Особенность |
|---|---|---|---|---|---|---|
| Bug | 🐛 | #ef4444 | 2 | Быстрый | Маленький | Базовый, ходит стаями |
| Glitch | ⚡ | #f97316 | 3 | Средний | Средний | Иногда телепортируется на 20px вперёд |
| Virus | 🦠 | #a855f7 | 5 | Медленный | Большой | Толстый, тяжело убить |
| Trojan | 🐴 | #eab308 | 4 | Средний | Средний | При смерти спавнит 2 маленьких Bug'а |
| Boss | 💀 | #dc2626 | 15 | Очень медленный | Очень большой | Появляется в последней волне |

### Появление по волнам

- Ранние волны: Bug, Glitch
- Средние волны: + Virus, Trojan
- Последняя волна: микс всех + Boss

## Башни

3 типа, каждая с 3 уровнями апгрейда.

| Башня | Эмодзи | Цвет | Цена | Урон | Скорость стрельбы | Радиус | Описание |
|---|---|---|---|---|---|---|---|
| Blaster | 🔮 | #3b82f6 | 30 | 1 | Быстрая (0.5с) | Средний | Одиночные выстрелы |
| Zapper | ⚡ | #8b5cf6 | 50 | 2 | Средняя (1с) | Большой | Молния по одному врагу |
| Cannon | 🎯 | #10b981 | 80 | 3 | Медленная (2с) | Маленький | AoE — бьёт всех в точке попадания |

### Апгрейды

| Уровень | Стоимость | Эффект |
|---|---|---|
| Lvl 1 | Базовая цена | Базовые характеристики |
| Lvl 2 | +40 монет | Урон ×1.5, радиус +15% |
| Lvl 3 | +70 монет | Урон ×2, радиус +30%, визуально больше + glow |

### Визуал башен на Canvas

- Скруглённый квадрат с цветной заливкой + эмодзи по центру
- Кольцо радиуса (пунктирный круг, видно при выделении)
- Lvl 2: чуть больше + двойная граница
- Lvl 3: ещё больше + glow-тень (shadowBlur: 15)
- Выстрел: линия от башни к врагу (цвет башни, fade out 200ms)
- Cannon AoE: круг взрыва 300ms

### Размещение

- 6-8 фиксированных слотов вдоль дороги
- Тап на пустой слот → подсвечивается, тап на башню в панели → размещается
- Тап на свою башню → popup "Улучшить" с ценой

## Визуальный стиль — Friendly Road

- Тёмный фон (#111118)
- Дорога: широкий roundRect путь с поворотами (#1e1e2e), пунктирная линия по центру
- Враги: скруглённые блоки с эмодзи + box-shadow
- Башни: яркие скруглённые квадраты с эмодзи + glow
- Пустые слоты: dashed border (#ffffff15) + "+"
- HUD: pill-shaped бейджи (монеты жёлтые, жизни красные, волна серая)
- Панель магазина: полупрозрачная полоса внизу с backdrop-blur

## UI экранов

### Фаза QUESTIONS

- Заголовок "Заработай монеты!" + текущий счётчик монет
- Вопросы по одному, стандартные компоненты
- "+30 ⭐" анимация при правильном ответе
- Прогресс-бар таймера сверху (15 сек)
- Итог: "Заработано: 90 ⭐" → кнопка "Строить башни"

### Фаза BUILD

- Canvas с полем, дорогой, слотами
- Панель 3 башен внизу (затемняются если не хватает монет)
- Тап на башню → popup улучшения
- HUD: монеты, жизни
- Кнопка "Начать волну ▶"

### Фаза BATTLE

- Canvas анимация: враги двигаются, башни стреляют
- HUD: монеты, жизни, номер волны
- Враг дошёл до конца → экран мигает красным
- Волна закончилась → переход в QUESTIONS

### Фаза RESULT

- Победа: "Защита пройдена!" + звёзды (3/2/1 по жизням)
- Поражение: "Базу захватили!" + кнопка "Попробовать снова"
- Кнопка "Продолжить" → `onAnswer(true/false)`

## Архитектура и файловая структура

```
src/components/games/tower-defense/
  TowerDefenseStep.tsx    — React обёртка, управляет фазами
  TDCanvas.tsx            — <canvas> компонент, рендер и game loop
  engine.ts               — игровая логика: tick(state, dt) → newState
  types.ts                — типы: Enemy, Tower, Projectile, Wave, GameState
  config.ts               — константы: цены, HP, скорости, цвета, эмодзи
  path.ts                 — дорога: массив точек, getPositionAlongPath(progress)
  questions.ts            — извлечение вопросов из steps урока
```

### Разделение ответственности

- **TowerDefenseStep.tsx** — React, управляет фазами (QUESTIONS/BUILD/BATTLE/RESULT), рендерит вопросы или canvas
- **TDCanvas.tsx** — canvas ref + requestAnimationFrame loop, принимает gameState, отдаёт события (towerPlaced, waveComplete, lifeLost)
- **engine.ts** — чистые функции без side effects, вся игровая логика
- **path.ts** — определение дороги и интерполяция позиции
- **questions.ts** — парсинг steps урока в формат вопросов для игры

### Интеграция

- Новый step type `"tower-defense"` в `courseApi.ts`
- Интерфейс: `{ type: "tower-defense" }` — вопросы берутся из соседних steps того же урока автоматически
- Добавляется case в `CourseStepPlayer.tsx` renderStep
- В `StepEditor.tsx` — простая форма (только выбор типа)

## Технические решения

- **Рендер:** HTML5 Canvas через `<canvas>` + `requestAnimationFrame`
- **Почему Canvas:** 0 зависимостей, работает в PWA и Capacitor, портируется в React Native через react-native-skia
- **Game loop:** `requestAnimationFrame` → `engine.tick(state, deltaTime)` → `canvas.draw(state)`
- **Состояние:** React state для фаз, plain object для игрового состояния (не React state — обновляется каждый кадр)
