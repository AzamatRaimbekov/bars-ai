# Landing Page — Bars AI

## Цель

Создать одностраничный лендинг для привлечения пользователей на платформу Bars AI с максимальной конверсией в бесплатную регистрацию.

## Целевая аудитория

- **Основная:** студенты и начинающие специалисты (B2C)
- **Вторичная:** компании, обучающие сотрудников (B2B)

## Тон и стиль

- Энергичный, геймифицированный (как Duolingo)
- Тёмная тема: `#000000` фон, `#0A0A0A` поверхности, `#F97316` оранжевый акцент
- Шрифт Inter, моноширинный JetBrains Mono для кода
- Без box-shadow — только borders и radial-gradient glows
- Анимации через Framer Motion
- Скруглённые углы: `rounded-2xl` для карточек, `rounded-xl` для кнопок

## Главное действие (CTA)

Кнопка "Начать бесплатно" → переход на `/register`.

## Структура (7 секций)

### Секция 1: Hero

- Надпись "Bars AI Platform" мелким текстом сверху
- Крупный заголовок: **"Учись. Играй. Побеждай."**
- Подзаголовок: "AI-ментор, интерактивные курсы и геймификация — всё в одном месте"
- Кнопка CTA: "Начать бесплатно"
- Плавающие бейджи: "+250 XP", "Level 5", "7 streak" — анимированные через Framer Motion
- Фоновый radial-gradient glow оранжевого цвета в правом верхнем углу

### Секция 2: Почему Bars AI (ключевые фичи)

Заголовок: **"Почему Bars AI?"**

Сетка 2×2 из карточек:

| Фича | Описание |
|------|----------|
| AI-ментор | Персональный помощник 24/7 |
| Python в браузере | Пиши код без установки |
| Геймификация | XP, лиги, бейджи, стрики |
| 44+ курсов | AI-рекомендации под тебя |

Каждая карточка: иконка + заголовок + краткое описание. Фон `rgba(255,255,255,0.04)`, border `rgba(255,255,255,0.06)`.

### Секция 3: Как это работает

Заголовок: **"Как это работает"**

3 шага с нумерованными иконками:

1. **Зарегистрируйся бесплатно** — Барсбек подберёт курсы по твоим интересам
2. **Проходи курсы и набирай XP** — интерактивные уроки, код в браузере, AI-помощник
3. **Соревнуйся и зарабатывай** — лиги, спринты с призами, рейтинги

Вертикальный layout, номера в оранжевых квадратах.

### Секция 4: Спринты — учись и зарабатывай

Заголовок: **"Спринты — учись и зарабатывай"**
Подзаголовок: "Регулярные соревнования с реальными призами"

Мокап карточки спринта:
- Название спринта + таймер "X дней осталось"
- 3 призовых места с суммами в тенге (50 000 ₸ / 30 000 ₸ / 15 000 ₸)
- Пояснение: "Набирай звёзды за выполнение уроков → поднимайся в рейтинге → забирай приз"

Фоновый glow жёлтого цвета (`rgba(251,191,36,0.1)`).

### Секция 5: Создавай курсы и зарабатывай

Заголовок: **"Создавай курсы. Зарабатывай."**
Подзаголовок: "Стань автором и получай доход с каждого ученика"

3 карточки-строки:
- **Удобный редактор** — создавай уроки с видео, кодом и квизами
- **AI генерирует за тебя** — опиши тему, AI создаст структуру курса
- **Зарабатывай** (зелёный акцент `#4ADE80`) — устанавливай цену и получай доход

### Секция 6: FAQ

Заголовок: **"Частые вопросы"**

3 вопроса в аккордеон-стиле:
- **Это бесплатно?** — Да, регистрация и большинство курсов бесплатны
- **Нужно устанавливать что-то?** — Нет, всё работает в браузере, включая Python
- **Как заработать на спринтах?** — Проходи уроки, набирай звёзды, попади в топ рейтинга — забирай приз

### Секция 7: Финальный CTA

- Заголовок: **"Готов начать?"**
- Подзаголовок: "Присоединяйся — это бесплатно"
- Крупная кнопка: "Создать аккаунт" → `/register`
- Фоновый оранжевый glow по центру
- Оранжевая граница вокруг секции для выделения

## Навигация (верхняя панель)

- Логотип "Bars AI" слева
- Ссылки: Фичи | Спринты | Авторам | FAQ (якорные ссылки на секции)
- Кнопки справа: "Войти" (ghost) + "Начать бесплатно" (primary)
- Фиксированная при скролле с backdrop-blur

## Технические решения

### Маршрутизация

- Новая страница `src/pages/Landing.tsx`
- Маршрут `/` для неаутентифицированных пользователей
- Аутентифицированные пользователи с `onboarding_complete` → `/dashboard`
- Аутентифицированные без onboarding → `/` (текущий онбординг)
- Обновить логику в `App.tsx`

### Компоненты

Создать в `src/components/landing/`:
- `HeroSection.tsx`
- `FeaturesSection.tsx`
- `HowItWorksSection.tsx`
- `SprintsSection.tsx`
- `CreateCoursesSection.tsx`
- `FAQSection.tsx`
- `FinalCTASection.tsx`
- `LandingNav.tsx`

### Анимации

- Framer Motion `motion.div` с `whileInView` для появления секций при скролле
- Плавающие бейджи в hero с `animate` (лёгкое покачивание)
- Smooth scroll при клике на якорные ссылки навигации

### Адаптивность

- Mobile-first: одна колонка, карточки фич 1×1
- Tablet (768px+): карточки фич 2×2
- Desktop (1024px+): hero с большим шрифтом, max-width контейнер 1200px

### SEO

- Мета-теги: title, description, og:image
- Семантическая разметка: `<header>`, `<main>`, `<section>`, `<footer>`
- `<h1>` на hero заголовке

## Что НЕ входит в scope

- Форма заявки для B2B (будет позже)
- Видео-превью платформы
- Интерактивное демо (подход 3 отклонён)
- Мультиязычность лендинга (только русский)
- A/B тестирование вариантов
