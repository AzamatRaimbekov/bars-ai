# Claude Code Advanced Course — Design Spec

**Date:** 2026-04-12
**Category:** AI-Assisted Development
**Difficulty:** intermediate-to-advanced
**Languages:** ru, en, ky (Kyrgyz)
**Total:** 18 sections, 89 lessons, ~1570 steps

---

## Overview

Большой продвинутый курс по Claude Code, охватывающий все аспекты работы с инструментом — от быстрого старта до мультиагентной оркестрации и реальных продакшн-кейсов. Модульная структура с практическими проектами в конце каждого модуля.

### Подход
- **Модульный с практическими проектами** — 4 модуля, каждый завершается проектом-кейсом
- **Насыщенные уроки** — 15-25 шагов в каждом уроке
- **Максимальное разнообразие шагов** — все 20+ типов с упором на terminal-sim и code-puzzle
- **Трёхъязычный контент** — ru/en/ky

### Типы шагов (приоритет использования)
1. **terminal-sim** — симуляция терминала, набор команд (основной тип)
2. **code-puzzle** — сборка кода из фрагментов (основной тип)
3. **quiz** — множественный выбор
4. **matching** — сопоставление пар
5. **flashcards** — карточки для запоминания
6. **drag-order** — расстановка в правильном порядке
7. **snippet-order** — упорядочивание фрагментов кода
8. **category-sort** — сортировка по категориям
9. **multi-select** — выбор нескольких правильных
10. **true-false** — правда/ложь
11. **fill_blanks** — заполни пропуски
12. **highlight-text** — выделение правильных сегментов
13. **type-answer** — ввод ответа вручную
14. **conversation-sim** — симуляция диалога
15. **info** — теоретический материал (markdown)
16. **video** — видео-контент

### Трёхъязычная поддержка

Текущая платформа поддерживает `{en, ru}`. Нужно расширить до `{en, ru, ky}`:
- Добавить `ky` во все интерфейсы `i18n`
- Обновить тип `LocalizedString` на `{en: string; ru: string; ky: string}`
- Добавить переключатель языка в UI
- Seed-файл курса будет содержать все три языка в каждом шаге
