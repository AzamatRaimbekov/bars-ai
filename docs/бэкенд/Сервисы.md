# Сервисы

Бизнес-логика в `backend/app/services/`. Роутеры вызывают сервисы, сервисы работают с БД.

## Список сервисов

| Файл | Назначение |
|---|---|
| `auth_service.py` | Регистрация, логин, хеширование, JWT |
| `user_service.py` | CRUD пользователей, профиль |
| `course_service.py` | Курсы, модули, уроки, теги, рекомендации; `create_course` и `update_course` передают/обновляют `tags`; `_course_to_dict` включает `tags` в ответ |
| `progress_service.py` | Прогресс, XP, уровни, стрики |
| `ai_service.py` | Генерация контента через Claude |
| `mentor_service.py` | AI-ментор, контекст чата |
| `quest_service.py` | Квесты и задания |
| `leaderboard_service.py` | Лидерборд (Redis sorted sets) |
| `trophy_service.py` | Трофеи за спринты |

## Паттерн

```python
# router → service → database
@router.post("/complete")
async def complete_lesson(data: LessonComplete, db: AsyncSession = Depends(get_db)):
    return await progress_service.complete(data, db)
```
