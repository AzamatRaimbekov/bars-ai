# Конфигурация

## Settings (`backend/app/config.py`)

Pydantic Settings, читает из `.env`:

```python
class Settings(BaseSettings):
    DATABASE_URL: str           # postgresql+asyncpg://...
    REDIS_URL: str              # redis://redis:6379/0
    ANTHROPIC_API_KEY: str      # sk-ant-...
    OPENAI_API_KEY: str         # опционально
    JWT_SECRET: str             # секрет для JWT
    JWT_ACCESS_EXPIRE_MINUTES: int = 30
    JWT_REFRESH_EXPIRE_DAYS: int = 7
    CORS_ORIGINS: str           # http://localhost:5173
    SMTP_HOST: str              # smtp.gmail.com
    SMTP_PORT: int              # 587
    SMTP_USER: str              # email для отправки
    SMTP_PASSWORD: str          # пароль / app password
    SMTP_FROM: str              # адрес отправителя
    FRONTEND_URL: str           # http://localhost:5173 (для ссылок в письмах)
```

## Redis (`backend/app/redis.py`)

Используется для:
- Лидерборд (sorted sets)
- Кэширование данных

## Docker

- `Dockerfile` — бэкенд (FastAPI + uvicorn)
- `Dockerfile.frontend` — фронтенд (Vite build → nginx)
- `docker-compose.yml` — в `backend/`, оркестрация

## Nginx

`nginx.conf` — проксирование `/api/` на бэкенд, остальное на SPA.
