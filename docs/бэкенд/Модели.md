# Модели

SQLAlchemy 2.0 async модели в `backend/app/models/`.

## Список

| Модель | Файл | Описание |
|---|---|---|
| User | `user.py` | Пользователь, пароль, direction, XP, уровень |
| Course | `course.py` | Курс с модулями и уроками |
| Progress | `progress.py` | Прогресс по курсам/урокам |
| Badge | `badge.py` | Достижения и бейджи |
| Sprint | `sprint.py` | Спринты и трофеи |
| Mentor | `mentor.py` | История чата с AI |
| RefreshToken | `refresh_token.py` | JWT refresh tokens |

## Миграции

```bash
cd backend
source venv/bin/activate

# Новая миграция
alembic revision --autogenerate -m "add field X to Y"

# Применить
alembic upgrade head

# Откатить
alembic downgrade -1
```
