# Структура проекта

## Корневая директория

```
bars-ai/
├── src/                    # Фронтенд (React SPA)
│   ├── App.tsx             # Корневой компонент, роутинг
│   ├── main.tsx            # Точка входа
│   ├── index.css           # Глобальные стили, Tailwind
│   ├── components/         # UI-компоненты
│   │   ├── chat/           # Чат-интерфейс
│   │   ├── courses/        # Карточки курсов
│   │   │   └── steps/      # PythonCodingStep и др.
│   │   ├── gamification/   # XP, badges, трофеи
│   │   ├── layout/         # Навбар, сайдбар, лейаут
│   │   ├── lesson/         # Плеер уроков
│   │   ├── mentor/         # AI-ментор
│   │   ├── roadmap/        # React Flow роадмап
│   │   ├── simulator/      # Симулятор собеседований
│   │   ├── sprint/         # Спринт-система
│   │   └── ui/             # Базовые UI-элементы
│   ├── pages/              # Страницы (20 шт.)
│   ├── store/              # Zustand сторы
│   ├── services/           # API-клиент
│   ├── hooks/              # React хуки (usePyodide и др.)
│   ├── lib/                # Утилиты (cn, constants, pyodideWorker)
│   └── types/              # TypeScript типы
│
├── backend/                # Бэкенд (FastAPI)
│   ├── app/
│   │   ├── main.py         # FastAPI app, CORS, роутеры
│   │   ├── config.py       # Pydantic Settings
│   │   ├── database.py     # SQLAlchemy async engine
│   │   ├── redis.py        # Redis pool
│   │   ├── models/         # SQLAlchemy модели
│   │   ├── routers/        # API эндпоинты (14 роутеров)
│   │   ├── services/       # Бизнес-логика
│   │   ├── schemas/        # Pydantic схемы
│   │   └── utils/          # Хелперы
│   ├── alembic/            # Миграции БД
│   ├── seed_*.py           # Скрипты заполнения курсов (40+)
│   └── requirements.txt    # Python-зависимости
│
├── admin/                  # Админ-панель (Vite + React + TS)
│   ├── src/
│   │   ├── App.tsx         # Роутинг админ-панели
│   │   ├── main.tsx        # Точка входа
│   │   ├── index.css       # Глобальные стили, Tailwind
│   │   ├── components/     # Layout, ProtectedRoute, StatusBadge, PageTransition
│   │   ├── pages/          # LoginPage, Dashboard, Users, Courses, Payments, Sprints
│   │   ├── store/          # authStore (Zustand)
│   │   └── lib/            # api.ts (apiFetch), cn.ts
│   ├── Dockerfile          # Nginx-based Docker image
│   ├── nginx.conf          # SPA fallback
│   ├── vite.config.ts      # Tailwind plugin, API proxy
│   └── package.json        # bars-ai-admin
│
├── docs/                   # Документация (Obsidian vault)
├── public/                 # Статические файлы
├── Dockerfile              # Backend Docker image
├── Dockerfile.frontend     # Frontend build
├── nginx.conf              # Nginx конфигурация
├── vite.config.ts          # Vite конфигурация
└── package.json            # Node.js зависимости
```

## Роутеры бэкенда

| Файл | Префикс | Назначение |
|---|---|---|
| `auth.py` | `/api/auth` | Регистрация, логин, refresh |
| `users.py` | `/api/users` | Профиль, обновление |
| `courses.py` | `/api/courses` | CRUD курсов, уроки |
| `progress.py` | `/api/progress` | Прогресс, XP, уровни |
| `ai.py` | `/api/ai` | AI-генерация контента |
| `mentor.py` | `/api/mentor` | AI-ментор чат |
| `quests.py` | `/api/quests` | Квесты, задания |
| `leaderboard.py` | `/api/leaderboard` | Лидерборд (Redis) |
| `leagues.py` | `/api/leagues` | Лиги пользователей |
| `sprint.py` | `/api/sprint` | Спринты, трофеи |
| `admin.py` | `/api/admin` | Админ-панель |
| `payments.py` | `/api/payments`, `/api/admin/payments` | Оплата курсов |
| `uploads.py` | `/api/uploads` | Загрузка и раздача файлов |
