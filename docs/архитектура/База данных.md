# База данных

## Модели (SQLAlchemy)

### User (`user.py`)
Основная модель пользователя — аутентификация, профиль, прогресс.

### Course (`course.py`)
Курсы с модулями и уроками. Связи с прогрессом.

### Progress (`progress.py`)
Прогресс пользователя по урокам, XP, уровни.

### Badge (`badge.py`)
Бейджи и достижения пользователей.

### Sprint (`sprint.py`)
Спринты (еженедельные челленджи), трофеи за выполнение.

### Mentor (`mentor.py`)
История чата с AI-ментором.

### RefreshToken (`refresh_token.py`)
JWT refresh-токены для безопасной аутентификации.

## Миграции

Используется **Alembic** с async-поддержкой.

```bash
# Создать миграцию
cd backend
alembic revision --autogenerate -m "описание"

# Применить миграции
alembic upgrade head
```

## Подключение

- Драйвер: `asyncpg` (async PostgreSQL)
- Engine: `SQLAlchemy 2.0` с `AsyncSession`
- Pool: стандартный пул asyncpg
