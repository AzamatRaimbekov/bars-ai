### Деплой

#### Автоматический (через GitHub)

Railway подключён к `AzamatRaimbekov/bars-ai`. Каждый push в `main` триггерит автоматический деплой.

```bash
git push origin main    # → автодеплой на Railway
```

#### Ручной (через CLI)

```bash
cd "/Users/azamat/Desktop/платформа обучнеи"
railway up             # деплой текущей директории
railway logs           # логи
railway status         # статус
```

### Dockerfile

Мультистейдж билд:
1. **Stage 1 (frontend):** `npm ci` → `npm run build` → `dist/`
2. **Stage 2 (backend):** `pip install` → копирует backend + `dist/` в `/app/static`
3. **CMD:** создание таблиц → seed → uvicorn на порту `$PORT`

### Миграции

Миграции запускаются через `create_tables.py` при старте контейнера (см. Dockerfile CMD).

Для ручного запуска:
```bash
railway run alembic upgrade head
```

### Мониторинг

- Health check: `GET /api/health` → `{"status": "ok"}`
- Логи: `railway logs` или Railway Dashboard
- Статус: `railway status`
