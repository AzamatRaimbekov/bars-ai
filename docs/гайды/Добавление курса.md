# Добавление курса

## Через seed-скрипт

В `backend/` уже есть 40+ seed-скриптов (`seed_*.py`). Каждый создаёт курс с модулями и уроками.

### Структура seed-скрипта

```python
# seed_new_course.py
import asyncio
from app.database import async_session
from app.models.course import Course, Module, Lesson

async def seed():
    async with async_session() as db:
        course = Course(
            title="Название курса",
            description="Описание",
            category="programming",
            difficulty="beginner",
            image_url="/images/course.png",
            author="PathMind"
        )
        db.add(course)
        await db.flush()

        module = Module(course_id=course.id, title="Модуль 1", order=1)
        db.add(module)
        await db.flush()

        lesson = Lesson(
            module_id=module.id,
            title="Урок 1",
            content="...",
            order=1
        )
        db.add(lesson)
        await db.commit()

asyncio.run(seed())
```

### Запуск

```bash
cd backend
source venv/bin/activate
python seed_new_course.py
```

## Через UI

Авторизованные пользователи могут создавать курсы через `/teach` и редактировать через `/teach/:id`.
