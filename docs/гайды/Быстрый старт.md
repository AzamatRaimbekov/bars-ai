# Быстрый старт

## Требования

- Node.js 18+
- Python 3.12+
- PostgreSQL
- Redis

## Установка

### Фронтенд

```bash
npm install
npm run dev          # http://localhost:5173
```

### Бэкенд

```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Настройка .env
cp .env.example .env   # заполнить переменные

# Миграции
alembic upgrade head

# Запуск
uvicorn app.main:app --host 0.0.0.0 --port 3847 --reload
```

### Оба одновременно

```bash
npm run dev:all      # concurrently: front + back
```

## Заполнение данных

```bash
cd backend
source venv/bin/activate
python seed_python_course.py    # один курс
# ... или любой другой seed_*.py
```
