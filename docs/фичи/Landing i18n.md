# Landing i18n

Многоязычная лендинг-страница с поддержкой 4 языков.

## Поддерживаемые языки

| Код | Метка |
|-----|-------|
| ru  | RU    |
| en  | EN    |
| ky  | KY    |
| uz  | UZ    |

## Реализация

Файл: `src/lib/landing-i18n.ts`

**Zustand-стор** `useLandingLang`:
- `lang` — текущий язык (по умолчанию `ru`)
- `setLang(lang)` — переключение; сохраняет выбор в `localStorage` с ключом `landing_lang`
- При инициализации читает сохранённое значение из `localStorage`

**Функция `t(key)`** — глобальная функция перевода:
```ts
export function t(key: string): string {
  const lang = useLandingLang.getState().lang
  return translations[lang]?.[key] ?? translations['ru'][key] ?? key
}
```
Fallback-цепочка: запрошенный язык → русский → сам ключ.

## Структура переводов

Объект `translations` содержит вложенные словари по кодам языков. Ключи сгруппированы по секциям:

- `nav.*` — навигация (Фичи, Форматы, Спринты, Авторам, FAQ, Войти)
- `hero.*` — героя (заголовок, подзаголовок, CTA)
- `features.*` — блок преимуществ
- `formats.*` — форматы обучения (tower defense, квизы, кодинг, matching, flashcards)
- `sprints.*` — секция спринтов
- `progression.*` — прогрессия и уровни
- `faq.*` — часто задаваемые вопросы

## Переключатель языка в навигации

Компонент навигации лендинга рендерит кнопки `LANG_LABELS` из `landing-i18n.ts`. При клике вызывает `useLandingLang.getState().setLang(code)`, что мгновенно перерисовывает весь лендинг без перезагрузки страницы.
