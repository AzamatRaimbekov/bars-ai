# Спринты

Соревновательная механика с денежными призами. Пользователи собирают трофеи за активность и соревнуются в лидерборде.

## Модели данных

`Sprint` (`backend/app/models/sprint.py`):
- `title`, `start_date`, `end_date`
- `status` — `active` / `completed` / `cancelled`
- `prizes` — JSON-массив с `place`, `amount`, `currency`, `bonus`
- `winners` — JSON с итогами после завершения

`TrophyEvent` — каждое начисление трофеев: `user_id`, `sprint_id`, `action`, `trophies`.

## Призовой фонд

Карточки призов отображает компонент `PrizeCard`. Структура по умолчанию:

| Место | Сумма   |
|-------|---------|
| 1     | $500    |
| 2     | $300    |
| 3     | $150    |

Бонус для 1-го места — процент от купленных курсов.

## API

Роутер `backend/app/routers/sprint.py`, префикс `/api/sprints`:

| Метод | Путь                     | Описание                        |
|-------|--------------------------|---------------------------------|
| GET   | `/active`                | Текущий активный спринт         |
| GET   | `/active/leaderboard`    | Лидерборд активного спринта     |
| GET   | `/my-trophies`           | Трофеи текущего пользователя    |

## Компоненты

- `SprintBanner` — баннер на дашборде с названием спринта, оставшимися днями и количеством трофеев пользователя. Ведёт на `/sprint`.
- `SprintLeaderboard` — отображает топ-3 в виде подиума (Crown/Trophy/Medal), остальных — строками. Текущий пользователь выделяется оранжевой полосой.
- `TrophyCounter` — кнопка в шапке с количеством трофеев спринта. Видна только если трофеев > 0.
- `PrizeCard` — карточка с медалью, суммой и бонусом.

## Управление

Создание, завершение и просмотр спринтов — через [[Admin Panel|Admin Panel]] (`/sprints`).
