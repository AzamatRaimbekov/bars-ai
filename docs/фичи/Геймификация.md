# Геймификация

Система мотивации, пронизывающая всю платформу. Связана с [[../бэкенд/Модели|моделями]] `Progress` и `UserBadge`.

## XP и уровни

Уровни определены в `src/lib/constants.ts`:

| Уровень       | XP порог |
|---------------|----------|
| Novice        | 0        |
| Apprentice    | 500      |
| Practitioner  | 1 500    |
| Expert        | 4 000    |
| Master        | 8 000    |
| Legend        | 15 000   |

Источники XP (`XP_REWARDS`):
- Завершение урока — 50 XP
- Идеальный квиз — 100 XP
- Симуляция интервью — 200 XP
- Дневной стрик — 25 XP

Прогресс отображается компонентом `XPBar` в шапке — показывает текущий уровень, прогресс-бар до следующего и суммарный XP.

## Стрики

Хранятся в `Progress.streak` и `Progress.longest_streak`. Компонент `StreakCounter` отображает иконку пламени с количеством дней. Вехи стрика (`STREAK_MILESTONES`): 7, 30, 100 дней.

При повышении уровня показывается анимированный `LevelUpModal` с иконкой трофея.

## Бейджи

5 степеней редкости с цветовой кодировкой (`BadgeReveal`):

| Редкость   | Цвет    |
|------------|---------|
| common     | #6B7280 |
| uncommon   | #22C55E |
| rare       | #3B82F6 |
| epic       | #A855F7 |
| legendary  | #F97316 |

Модель `UserBadge` хранит `badge_id` (строка), `user_id` и `earned_at`. Уникальный составной индекс предотвращает дублирование. При получении бейджа воспроизводится анимация `BadgeReveal` с пружинным вращением.

## Ежедневные квесты

Компонент `DailyQuests` загружает задания через `questApi.getDailyQuests()` и обновляет их каждые 30 секунд.

Типы квестов: `complete_lessons_1/3/5`, `earn_xp_50/100/200`, `maintain_streak`, `complete_quiz_2`, `study_30min`, `complete_node`. Каждый квест содержит `progress`, `target` и `xp_reward`.

## Лиги

Поле `Progress.league` хранит текущую лигу пользователя (bronze, silver, gold и т.д.). Лиги сбрасываются еженедельно на основе `xp_this_week`.
