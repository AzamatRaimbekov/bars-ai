# AI Менторство

Персональный AI-ассистент Барсбек доступен на всей платформе. Реализован в `backend/app/services/ai_service.py`.

## Модель

Используется `claude-sonnet-4-20250514` через Anthropic API (`https://api.anthropic.com/v1/messages`). Таймаут: 60 с для обычных запросов, 180 с для генерации курсов (max_tokens > 4000).

## 8 специализаций (MENTOR_PROMPTS)

| Ключ          | Специализация                                                              |
|---------------|----------------------------------------------------------------------------|
| `frontend`    | HTML, CSS, JavaScript, TypeScript, React, веб-разработка                   |
| `english`     | Английский язык, грамматика, произношение, A1–C2                           |
| `callcenter`  | Колл-центр, клиентский сервис, разрешение конфликтов, скрипты              |
| `cib`         | Corporate & Investment Banking, финансы, Excel-моделирование               |
| `programming` | Python, алгоритмы, структуры данных, отладка                               |
| `design`      | UI/UX, типографика, цветовая теория, Figma                                 |
| `marketing`   | Digital-маркетинг, SMM, таргетинг, копирайтинг                             |
| `languages`   | Изучение любого языка, мнемоники, культурный контекст                      |

Базовый промпт (`BARSBEK_BASE`) описывает Барсбека как доброго старшего брата, который искренне хочет помочь ученику.

## Функции сервиса

- `chat(messages, direction, language)` — диалог с учётом специализации. При `language="ru"` добавляется инструкция отвечать на русском.
- `assess(direction, answers)` — оценка уровня по ответам; возвращает `beginner` / `intermediate` / `advanced`.
- `tip(direction, level, user_id)` — совет дня, кешируется в Redis на 24 часа.
- `score(question, answer, direction)` — оценка ответа на интервью (1–10) + обратная связь + эталонный ответ. Возвращает JSON.
- `transcribe(audio_bytes)` — транскрипция через OpenAI Whisper (`whisper-1`), используется для голосового режима.

## Голосовой режим

Аудио записывается на фронтенде и отправляется на `POST /api/ai/transcribe`. Ответ Whisper содержит `text` и `confidence`. После транскрипции текст передаётся в обычный `chat`.

## Кеширование

Совет дня (`tip`) кешируется в Redis с ключом `tip:{user_id}:{date}` и TTL 86400 секунд.
