# Курсы и шаги

## Структура курса

```
Course
  └── Section (позиция, название)
        └── Lesson (позиция, xp_reward, content_type)
              └── Step[] (тип + данные)
```

Интерфейсы определены в `src/services/courseApi.ts`: `CourseCard`, `CourseDetail`, `CourseSection`, `CourseLesson`, `LessonStep`.

Курс также может иметь `roadmap_nodes` и `roadmap_edges` для отображения в виде карты прогресса.

## Типы шагов (StepType)

Полный union-тип из `courseApi.ts` — 28 типов:

**Информация и медиа**
- `info` — текст в Markdown
- `video` — видеоплеер
- `audio` — аудиоплеер
- `embed` — встраиваемый контент
- `resources` — список ресурсов/ссылок

**Тесты и опросы**
- `quiz` — выбор одного варианта
- `multi-select` — несколько правильных ответов
- `true-false` — верно/неверно
- `type-answer` — ввод ответа вручную

**Карточки и память**
- `flashcards` — карточки с переворотом
- `matching` — сопоставление пар
- `category-sort` — сортировка по категориям
- `timeline` — расстановка событий по хронологии

**Работа с текстом**
- `fill-blank` — вставка пропущенных слов
- `highlight-text` — выделение нужных фрагментов
- `cloze-passage` — заполнение пропусков в тексте
- `drag-order` — перетаскивание для упорядочивания
- `snippet-order` — выстраивание фрагментов кода

**Язык**
- `pronunciation` — произношение (через микрофон)
- `listening-comprehension` — аудирование
- `word-builder` — составление слов
- `sentence-translation` — перевод предложения
- `conversation-sim` — симуляция диалога

**Код**
- `code-puzzle` — сборка кода из фрагментов
- `code-editor` — полноценный редактор кода
- `terminal-sim` — симуляция терминала
- `python-coding` — запуск Python через Pyodide в браузере (см. [[../фронтенд/Python Coding|Python Coding]])

**Геймификация**
- `tower-defense` — tower defense мини-игра с вопросами между волнами
- `image-hotspot` — клик по областям изображения

## AI-генерация курсов

Администратор может сгенерировать курс через [[Admin Panel|Admin Panel]] (`/ai-course`). Параметры: тема, промпт, язык, сложность, количество секций, загружаемые файлы (контекст). Бэкенд вызывает `POST /api/ai/generate-course` с `multipart/form-data`.
