# Компоненты

Все компоненты в `src/components/`, организованы по доменам.

## Структура

| Директория | Назначение |
|---|---|
| `chat/` | Чат-интерфейс (сообщения, ввод, bubble) |
| `courses/` | Карточки курсов, списки, фильтры, ступени уроков |
| `courses/steps/` | Компоненты ступеней: PythonCodingStep, ClozePassageStep, WordBuilderStep и др. |
| `dance/` | Танцевальный модуль (MediaPipe) |
| `games/` | Мини-игры для обучения |
| `gamification/` | XP-бар, бейджи, трофеи, стрики |
| `layout/` | Навбар, сайдбар, MobileNav, общий лейаут |
| `lesson/` | Плеер уроков (видео, текст, код, квиз) |
| `mentor/` | AI-ментор виджет |
| `roadmap/` | React Flow роадмап-визуализация |
| `simulator/` | Симулятор собеседований |
| `sprint/` | Спринт-баннер, SprintCard, TrophyCounter |
| `ui/` | Базовые элементы (Button, Input, Card, Modal) |

## Страницы (20 шт.)

Все в `src/pages/`. Каждая страница — отдельный файл.

Ключевые:
- **Dashboard** — главная после логина, виджеты прогресса
- **Courses** — каталог 40+ курсов; фильтры по категории, сложности, цене и тегам; отображение тегов на карточках курсов
- **CourseEditor** (`/teach/:id`) — редактор курса; вкладка «Информация» содержит редактор тегов (chips + ввод по Enter), теги сохраняются в поле `tags: string[]` и отправляются при сохранении курса
- **CourseLearn** — плеер урока с шагами
- **Mentor** — чат с AI-ментором
- **Sprint** — еженедельные спринты с трофеями
- **Admin** — админ-панель (статистика, управление)

## Админ-панель (admin/src/components/)

| Компонент | Назначение |
|---|---|
| `Layout.tsx` | Сайдбар (240px) + топбар + Outlet для контента |
| `ProtectedRoute.tsx` | Редирект на /login если нет токена |
| `PageTransition.tsx` | Framer Motion анимация (fade + slide) |
| `StatusBadge.tsx` | Цветные бейджи (green/yellow/red/gray/purple) |
