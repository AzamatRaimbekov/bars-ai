# Роутинг

React Router v7 с `BrowserRouter`.

## Маршруты

| Путь | Страница | Доступ |
|---|---|---|
| `/login` | Login | Публичный |
| `/register` | Register | Публичный |
| `/` | Onboarding (4-step wizard с Barsbek) | AuthGuard |
| `/dashboard` | Dashboard | AuthGuard |
| `/roadmap` | Roadmap | AuthGuard |
| `/mentor` | Mentor | AuthGuard |
| `/simulator` | Simulator | AuthGuard |
| `/lesson/:id` | Lesson | AuthGuard |
| `/achievements` | Achievements | AuthGuard |
| `/leaderboard` | Leaderboard | AuthGuard |
| `/leagues` | Leagues | AuthGuard |
| `/profile` | Profile | AuthGuard |
| `/courses` | Courses | Публичный |
| `/courses/:id` | CourseDetail | Публичный |
| `/courses/:id/roadmap` | CourseRoadmap | AuthGuard |
| `/courses/:id/learn/:lessonId` | CourseLearn | AuthGuard |
| `/teach` | Teach | AuthGuard |
| `/teach/:id` | CourseEditor | AuthGuard |
| `/sprint` | Sprint | AuthGuard |
| `/admin` | Admin | AuthGuard |

## Админ-панель (admin/)

Отдельное Vite-приложение в `admin/`. React Router v7, BrowserRouter.

| Путь | Страница | Доступ |
|---|---|---|
| `/login` | LoginPage | Публичный |
| `/` | Dashboard | AdminGuard |
| `/users` | UsersPage | AdminGuard |
| `/courses` | CoursesPage | AdminGuard |
| `/payments` | PaymentsPage | AdminGuard |
| `/sprints` | SprintsPage | AdminGuard |

AdminGuard проверяет `bars_admin_token` в sessionStorage.
